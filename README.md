# R Shiny Workshop for PhD Students

This repository contains the materials for a practical **R Shiny workshop for PhD students**. The workshop is designed as a hands-on introduction to building interactive web applications in R, with a strong focus on learning by doing.

## Repository contents

- **Presentation**  
  The slide deck used during the workshop.

- **Assignment files**  
  Practical exercises for students to complete during or after the session.



## Workshop focus

During this workshop, students learn how to:

- create a basic R Shiny application
- understand the structure of a Shiny app
- build user interfaces with `fluidPage()`, `fluidRow()`, and `column()`
- display structured content in a web application
- add simple styling
- work with static and dynamic resume examples
- use reactive input fields in a more advanced Shiny app

## Assignments

The repository includes main resume-based assignments:

### 1. Static resume application
Students create a fully written resume directly in Shiny code.  
This assignment focuses on:

- Shiny layout structure
- text elements such as headers and paragraphs
- lists and sections
- simple styling
- adding an image and external links

### 2. Dynamic resume generator
Students build a more advanced app in which the resume updates based on user input.  
This assignment focuses on:

- input widgets
- reactive programming
- rendering text and lists dynamically
- conditional display of content
- exporting resume content

## Learning objectives

After completing this workshop, students should be able to:

- understand the basic principles of R Shiny
- build a small interactive web app independently
- structure content clearly in a dashboard-like layout
- distinguish between static and reactive Shiny applications
- apply Shiny to communicate academic or professional information
